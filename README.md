# Scikit-* Ecosystem Overview

This repository provides a structured, from-beginner-to-advanced overview of the **scikit-\*** ecosystem.  
Each chapter introduces one part of the ecosystem, with clear explanations and minimal runnable code examples.

This map is designed to help learners understand:
- How different scikit-\* packages relate to each other  
- What tasks each package can solve  
- Typical workflows combining multiple scikit tools  
- How to gradually deepen understanding chapter by chapter  

---

## Ecosystem Map

Below is a high-level grouping of major scikit-\* projects by domain.

### **Core ML (Machine Learning)**
- **scikit-learn** ‚Äî Classic, general-purpose machine learning library.
- **scikit-optimize (skopt)** ‚Äî Bayesian optimization and hyperparameter search.
- **scikit-survival** ‚Äî Survival analysis built on top of scikit-learn.

### **Image Processing**
- **scikit-image** ‚Äî Algorithms for image filtering, segmentation, feature extraction.
- **scikit-video** ‚Äî Basic tools for reading and processing video data.

### **Signal & Time Series**
- **scikit-signal** ‚Äî Signal-processing utilities.
- **scikit-dsp-comm** ‚Äî Digital communication + DSP functions.

### **Scientific Computing**
- **scikit-bio** ‚Äî Bioinformatics and microbiome analytics.
- **scikit-hep** ‚Äî High-energy physics data modeling and analysis.
- **scikit-fem** ‚Äî Finite Element Method (FEM) solvers.

### **Other Specialized Domains**
- **scikit-gstat** ‚Äî Geostatistics, variograms, kriging.
- **scikit-rf** ‚Äî Radio frequency and microwave engineering.
- **scikit-mobility** ‚Äî Mobility and trajectory data analysis.
- **scikit-tda** ‚Äî Topological data analysis.

## Chapter Structure (Learning Path)

Each chapter stands alone ‚Äî you can start anywhere ‚Äî but the difficulty increases gradually.

1. **Core ML with scikit-learn**  
2. **Model selection & Bayesian optimization with skopt**  
3. **Image processing with scikit-image**  
4. **Signal & time-series analysis**  
5. **Domain-specific scikit kits**  
6. **Project patterns and practical workflows**  
7. **Advanced scikit-learn (Pipelines, Stacking, Feature Selection)**
